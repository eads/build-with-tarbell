## Install Tarbell

<hr />

#### OS X

<pre><code class="bash" data-trim>
sudo easy_install pip
sudo pip install tarbell==0.9b4
</code></pre>

<hr/>

#### Ubuntu

<pre><code class="bash" data-trim>
sudo apt-get install python-pip
sudo pip install tarbell==0.9b4
</code></pre>

<hr/>

#### Windows

<pre>Sorry, Windows users! Try <a href="http://vagrantup.com">Vagrant</a>.</pre>
